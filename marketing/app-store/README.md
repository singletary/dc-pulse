# App Store assets

This directory contains the canonical App Store screenshot package for DC Pulse 1.0.

## Generate

Install or otherwise make `sharp` available to Node, then run:

```sh
node marketing/app-store/generate-screenshots.mjs
```

The generator reads the privacy-reviewed captures in `source` and writes the final files to:

- `screenshots/en-US/iPhone-6.9`
- `screenshots/en-US/iPhone-6.5`

Each capture is composited using the checked-in iPhone 17 Pro display alpha mask in
`device-masks`. The mask was generated by Apple’s iOS Simulator and preserves its
native continuous four-corner display geometry. The surrounding device presentation
uses proportions measured from the Simulator’s iPhone 17 Pro window, including its
display-to-body ratio, restrained bezel, and side-control placement.

## Output contract

- 1320 × 2868 pixels for the 6.9-inch portrait slot
- 1284 × 2778 pixels for the 6.5-inch portrait slot
- PNG without an alpha channel
- English (U.S.)
- Ordered by the numeric filename prefix

The source captures are staged at a well-known public Downtown DC landmark. Do not replace them with captures from a developer or tester's actual location, saved Home, email, or other personal data.
